## Run Locally

Clone the project

```bash
  git clone https://github.com/BidyasagarAnupam/top-courses-react-app.git
```

Go to the project directory

```bash
  cd [name of the folder]
```

Install dependencies

```bash
  npm install or npm i
```

Start the server

```bash
  npm run start or npm start
```
